# GraphML Viewer

一个纯前端的 `.graphml` 文件可视化页面，直接在浏览器中打开 `graphml简易查看器.html` 即可使用，无需额外依赖或构建步骤。

## 功能概览
- **文件上传/默认示例**：支持本地上传 `.graphml`/`.xml` 文件，未上传时会自动加载内置示例数据。
- **中英文切换**：内置语言选择器，界面文案、统计信息与提示同步切换（中文/English）。
- **布局与过滤**：提供力导向、按年份/频率的径向布局，并可一键过滤孤立节点，仅保留主干网络。
- **标签与大小控制**：可按频率阈值或 Top-N 显示标签，支持相对/绝对节点大小、基础尺寸与排斥力调节。
- **交互提示**：节点悬停显示作者、年份、频率、期刊等信息，并展示年份渐变图例。

## 使用步骤
1. 双击或通过浏览器打开仓库中的 `graphml简易查看器.html` 文件。
2. 左侧控制面板可调整语言、布局、标签、节点大小、排斥力等参数；勾选“仅显示主干网络”可过滤孤岛节点。
3. 通过“数据源”上传本地 `.graphml` 文件，或直接使用默认示例数据进行体验。
4. 在画布上可拖动、缩放视图，鼠标悬停节点查看详细信息，点击“重置视图”恢复初始缩放。

## 提示
- 本项目完全在浏览器本地运行，不会上传数据到网络。
- 如需修改默认语言，将 `graphml简易查看器.html` 中 `currentLang` 的初始值从 `zh` 改为 `en` 即可。
